Class Void

Void's role in Atom world is same as undefined & null 's role in normal JavaScript world. It is designed to replace values that can be converted to false, such as undefined & null. So falsy values can flowing through the Atoms as normal values flowing. For the typical usages, please check nilToVoidT & defaultT.
