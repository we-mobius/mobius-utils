Class Terminator

Terminator is designed as a signal for "interruption" of Atom Flow. Mutation will not mutate (trigger an transformation to update the downstream data's value) a Data or Datar which value is Terminator. Data will not mutate (update own value to income transformation's result) a Mutation or Mutator or Transformation which result is Terminator. For the typical usages, please check filterT or skipT or takeT.
